# URP-IP21: Crisis Resource Triage

**Type:** Intelligence Pattern (Tier 2)
**Function:** Rapidly allocating scarce resources during crisis
**Domain Independence:** Universal across any crisis context

## Pattern Definition

### Core Procedure

1. **Demand Assessment**: What needs attention? How urgent is each?
2. **Resource Inventory**: What resources are available?
3. **Triage Decision**: What gets resources now vs. later vs. never?
4. **Allocation Execution**: Assign resources to prioritized needs
5. **Continuous Reprioritization**: Adjust as situation evolves

### Expert Heuristic

> "In crisis, you can't save everything. Triage ruthlessly—apply resources where they'll have the most impact, and accept that some things won't get attention."
