# URP-IP26: Message Sequencing Strategy

**Type:** Intelligence Pattern (Tier 2)
**Function:** Sequencing communications for cumulative effect
**Domain Independence:** Universal across any persuasion/communication context

## Pattern Definition

### Core Procedure

1. **End-State Definition**: What's the desired final position/decision?
2. **Starting Point Assessment**: Where is the audience now?
3. **Waypoint Identification**: What intermediate positions are needed?
4. **Message Sequence Design**: What messages move audience through waypoints?
5. **Timing Calibration**: When should each message be delivered?

### Expert Heuristic

> "Persuasion is rarely a single message—it's a sequence. Design the journey from where they are to where you need them to be."
