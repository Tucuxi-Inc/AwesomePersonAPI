# URP-IP15: Stakeholder Assessment Validation

**Type:** Intelligence Pattern (Tier 2)
**Function:** Validating stakeholder understanding across perspectives
**Domain Independence:** Universal across any multi-stakeholder context

## Pattern Definition

### Core Procedure

1. **Assessment Articulation**: What do I think each stakeholder's position/interests are?
2. **Validation Method Selection**: How can I validate my understanding?
3. **Cross-Check Execution**: Test understanding through questions, actions, third parties
4. **Discrepancy Identification**: Where was my assessment wrong?
5. **Model Update**: Update stakeholder model based on validation

### Expert Heuristic

> "Your model of stakeholders is only useful if it's accurate. Test it before you rely on it."
