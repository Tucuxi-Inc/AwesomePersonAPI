# URP-IP14: Competitive Position Assessment

**Type:** Intelligence Pattern (Tier 2)
**Function:** Assessing relative position in competitive landscape
**Domain Independence:** Universal across any competitive context
**Orchestration Role:** Provides strategic position awareness for competitive decisions

## Pattern Definition

### Core Procedure

1. **Position Mapping**: Where do I stand relative to competitors/alternatives?
2. **Strength/Weakness Analysis**: What are relative advantages and disadvantages?
3. **Sustainability Assessment**: Which advantages are sustainable vs. temporary?
4. **Gap Analysis**: Where are the critical gaps to address?
5. **Position Strategy**: How should position inform strategy?

### Expert Heuristic

> "Know where you stand before deciding where to compete. Fight battles where you have advantage; avoid battles where you don't."

## Evidence from Literature

- **Competitive Strategy** (Porter): Positioning and competitive advantage
- **SWOT Analysis**: Structured position assessment
- **Resource-Based View**: Sustainable competitive advantage
- **Blue Ocean Strategy**: Competitive space analysis
