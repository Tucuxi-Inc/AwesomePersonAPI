# URP-IP16: Cross-Functional Decision Coordination

**Type:** Intelligence Pattern (Tier 2)
**Function:** Coordinating decisions across organizational boundaries
**Domain Independence:** Universal across any multi-function context

## Pattern Definition

### Core Procedure

1. **Function Mapping**: What functions/parties need to be involved in this decision?
2. **Process Understanding**: What are each function's decision processes and timelines?
3. **Coordination Design**: How do I synchronize across functions?
4. **Conflict Management**: How do I handle cross-functional disagreements?
5. **Execution Tracking**: How do I ensure coordinated execution?

### Expert Heuristic

> "Cross-functional decisions fail when each function optimizes for itself. Coordination means getting everyone to optimize for the whole."
