# URP-IP6: Portfolio Resource Optimization

**Type:** Intelligence Pattern (Tier 2)
**Function:** Optimizing resource allocation across portfolio
**Domain Independence:** Universal across any multi-initiative context
**Orchestration Role:** Ensures resources flow to highest-value uses across portfolio

## Pattern Definition

### Core Procedure

1. **Portfolio Inventory**: What initiatives/opportunities compete for resources?
2. **Value Assessment**: What's the expected value of each portfolio element?
3. **Resource Mapping**: What resources are available? What does each element require?
4. **Optimization**: How should resources be allocated for maximum portfolio value?
5. **Rebalancing**: How do I adjust allocation as circumstances change?

### Expert Heuristic

> "The goal isn't to maximize each initiative—it's to maximize the portfolio. Sometimes the best allocation starves good opportunities to feed great ones."

## Evidence from Literature

- **Portfolio Theory** (Markowitz): Optimization across investments
- **Resource Allocation**: Operations research approaches
- **Opportunity Cost**: Allocation decisions include foregone alternatives
- **Dynamic Rebalancing**: Continuous portfolio optimization
