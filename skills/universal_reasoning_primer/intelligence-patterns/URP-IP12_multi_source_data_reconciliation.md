# URP-IP12: Multi-Source Data Reconciliation

**Type:** Intelligence Pattern (Tier 2)
**Function:** Reconciling conflicting information from multiple sources
**Domain Independence:** Universal across any information synthesis context
**Orchestration Role:** Produces coherent understanding from messy, conflicting data

## Pattern Definition

### Core Procedure

1. **Source Inventory**: What sources provide information on this topic?
2. **Source Evaluation**: How reliable is each source? What biases exist?
3. **Conflict Identification**: Where do sources conflict?
4. **Reconciliation**: How do I resolve conflicts? Which sources to weight more?
5. **Synthesis**: What's the best integrated understanding given all sources?

### Expert Heuristic

> "When sources conflict, don't just pick a side. Ask: why do they conflict? The answer often reveals more than either source alone."

## Evidence from Literature

- **Intelligence Analysis** (Heuer): Evaluating and reconciling source information
- **Source Criticism**: Historical method for evaluating sources
- **Triangulation**: Using multiple sources to validate
- **Bayesian Updating**: Integrating multiple evidence sources
