# URP-MC10: Attention Allocation

**Type:** Meta-Cognitive Pattern (Tier 3)
**Function:** Directing focus to high-value targets; managing attentional resources
**Domain Independence:** Universal across all cognitively demanding tasks
**Orchestration Role:** Controls where limited attention goes and when to shift

## Pattern Definition

### Trigger Condition
Situations involving:
- Multiple competing demands for attention
- Need to sustain focus on complex task
- Distractions threatening priority work
- Uncertainty about what deserves attention
- Need to shift between focused and diffuse modes

### Core Procedure

1. **Priority Assessment**
   - What's most important to attend to right now?
   - What has highest value per attention-unit?
   - What will suffer most if ignored?
   - What's urgent vs. what's important?

2. **Attention Mode Selection**
   - Focused: deep concentration on single target
   - Scanning: broad awareness across field
   - Alternating: switching between targets
   - Divided: parallel processing (limited)

3. **Distraction Management**
   - What's competing for attention?
   - Is this distraction valuable or noise?
   - How do I protect focus from interruption?
   - Should I attend to interruption or defer it?

4. **Attention State Monitoring**
   - Is my attention where I intend it to be?
   - Am I drifting or staying on target?
   - Is my focus sharp or diffuse?
   - Am I maintaining or fatiguing?

5. **Dynamic Reallocation**
   - When should I shift attention?
   - Has priority changed?
   - Is current target depleted of value?
   - Does new information warrant attention shift?

### Expert Heuristic

> "Attention is the scarcest resource—guard it more jealously than time. The question is never 'can I attend to this?' but 'is this the highest-value use of my attention right now?' Most 'attention problems' are priority problems in disguise."

## Evidence from Literature

### Example 1: Attention as Resource
**Context:** Kahneman (1973) attention and effort
**Evidence:** Attention is a limited capacity that must be allocated strategically
**Insight:** Attention allocation is a resource management problem

### Example 2: Executive Attention
**Context:** Engle (2002) working memory and attention control
**Evidence:** Individual differences in attention control predict complex cognition performance
**Insight:** Attention control is a trainable skill

### Example 3: Mind Wandering
**Context:** Schooler et al. (2011) meta-awareness of attention
**Evidence:** People often don't notice when attention drifts; meta-awareness improves control
**Insight:** Monitoring attention state is prerequisite to controlling it

## Decision Criteria

**Criterion 1: Value Assessment**
- IF high-value target available, THEN focus attention there
- IF all targets low-value, THEN consider whether to disengage
- Rationale: Attention should track value

**Criterion 2: Mode Selection**
- IF deep processing needed, THEN focused mode
- IF surveying for opportunities/threats, THEN scanning mode
- IF multiple equally important targets, THEN alternating mode
- Rationale: Match attention mode to task demands

**Criterion 3: Distraction Evaluation**
- IF distraction is higher value than current focus, THEN shift
- IF distraction is lower value, THEN protect focus
- IF uncertain, THEN quick evaluation then return
- Rationale: Not all interruptions are costs

**Criterion 4: Fatigue Response**
- IF attention quality degrading, THEN consider break or switch
- IF forced to continue, THEN accept lower quality
- Rationale: Depleted attention is not efficient attention

## Contrast with Naive Approaches

**Naive Approach**: Attend to whatever is loudest or most recent.
**Expert Approach**: Attend to whatever is highest value regardless of salience.

**Naive Approach**: Try to attend to everything at once.
**Expert Approach**: Accept attention limits; prioritize ruthlessly.

**Naive Approach**: All distractions are bad; block everything.
**Expert Approach**: Evaluate distractions; some warrant attention shift.

**Naive Approach**: Push through attention fatigue; effort conquers all.
**Expert Approach**: Manage attention as renewable resource; schedule recovery.
