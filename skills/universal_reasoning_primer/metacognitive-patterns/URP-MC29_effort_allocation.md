# URP-MC29: Effort Allocation

**Type:** Meta-Cognitive Pattern (Tier 3)
**Function:** Distributing cognitive effort optimally across demands
**Domain Independence:** Universal across all cognitively demanding contexts
**Orchestration Role:** Master resource allocation operator; governs investment of mental effort

## Pattern Definition

### Trigger Condition
Situations involving:
- Multiple tasks competing for limited mental energy
- Decisions about how hard to work on something
- Need to sustain performance over extended periods
- Tradeoffs between thoroughness and coverage
- Strategic decisions about where to invest

### Core Procedure

1. **Effort Budget Assessment**
   - How much cognitive capacity do I have available?
   - What's my current depletion level?
   - What demands are coming that I need to reserve for?
   - What's the time horizon?

2. **Value Mapping**
   - Which tasks have highest value-per-effort?
   - Where does marginal effort produce most return?
   - What's the minimum viable effort for each task?
   - What's the point of diminishing returns?

3. **Allocation Decision**
   - How much effort to invest in each task?
   - Where to front-load vs. back-load effort?
   - What to do adequately vs. excellently?
   - What to decline or defer?

4. **Effort Monitoring**
   - Am I spending effort as planned?
   - Is the return matching investment?
   - Should I reallocate mid-stream?
   - Am I depleting faster than expected?

5. **Recovery Planning**
   - When will I recover depleted capacity?
   - What activities restore vs. deplete effort capacity?
   - How do I protect peak capacity for priority demands?

### Expert Heuristic

> "Effort is finite and renewable but not unlimited. The goal isn't to work hard at everything—it's to work hard at the right things and adequately at the rest. Maximum effort on everything means inadequate effort on priorities."

## Evidence from Literature

### Example 1: Region of Proximal Learning
**Context:** Metcalfe & Kornell (2005) study allocation
**Evidence:** Optimal learners allocate more effort to items in the "region of proximal learning"
**Insight:** Effort allocation should target where effort can make difference

### Example 2: Self-Control as Resource
**Context:** Baumeister (2002) ego depletion research
**Evidence:** Self-control and cognitive effort draw on limited resource
**Insight:** Effort conservation is rational, not lazy

### Example 3: Effort Investment
**Context:** Kahneman (1973) attention and effort
**Evidence:** Mental effort has costs; rational agents minimize effort for given outcomes
**Insight:** Effort minimization is normative, not pathological

## Decision Criteria

**Criterion 1: Marginal Value**
- IF marginal effort would produce high value, THEN invest
- IF marginal effort would produce little value, THEN conserve
- Rationale: Effort follows value

**Criterion 2: Capacity Protection**
- IF high-value demands upcoming, THEN reserve capacity
- IF no upcoming demands, THEN can afford to deplete
- Rationale: Strategic conservation enables peak performance

**Criterion 3: Minimum Viable**
- IF task requires completion but isn't high-value, THEN minimum viable effort
- IF task is high-value, THEN invest fully
- Rationale: Not everything deserves maximum effort

**Criterion 4: Recovery Integration**
- IF depleted, THEN either recover or accept lower quality
- IF recovery time available, THEN plan effort around it
- Rationale: Depletion is temporary but must be managed

## Contrast with Naive Approaches

**Naive Approach**: Maximum effort on everything; "always give 100%."
**Expert Approach**: Strategic effort allocation; give 100% to the right things.

**Naive Approach**: Effort allocation is laziness management.
**Expert Approach**: Effort allocation is strategic resource optimization.

**Naive Approach**: Work until capacity is exhausted.
**Expert Approach**: Manage capacity strategically; reserve for priorities.

**Naive Approach**: Effort is free; spend freely.
**Expert Approach**: Effort is scarce; invest wisely.
