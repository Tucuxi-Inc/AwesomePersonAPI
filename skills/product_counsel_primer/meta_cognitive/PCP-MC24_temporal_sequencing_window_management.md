# PCP-MC24: Temporal Sequencing & Window Management

## Pattern Definition

### Trigger Condition
Activate when the ORDER and TIMING of actions significantly affects outcomes—where doing the same things in different sequence produces different results, or where windows of opportunity open and close.

### Core Procedure
1. **Dependency Mapping**: Identify which actions depend on which others
2. **Window Detection**: Find time-sensitive opportunities and their triggers
3. **Sequence Optimization**: Order actions to maximize option value and minimize wasted effort
4. **Momentum Assessment**: Evaluate and manage process momentum
5. **Reversibility Analysis**: Distinguish reversible from irreversible commitments
6. **Deadline Architecture**: Structure deadlines to create useful pressure

### Expert Heuristic
"Sequence is strategy. The expert knows that asking permission before acting creates different dynamics than acting then asking forgiveness—and chooses the sequence that best serves the ultimate goal."
