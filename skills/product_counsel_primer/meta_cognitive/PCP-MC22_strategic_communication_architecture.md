# PCP-MC22: Strategic Communication Architecture

## Pattern Definition

### Trigger Condition
Activate when communication must reach multiple audiences through multiple channels with different framing, timing, and depth—while maintaining consistency and strategic coherence.

### Core Procedure
1. **Audience Segmentation**: Identify distinct audiences with different needs and contexts
2. **Channel Mapping**: Match channels to audiences and message types
3. **Message Versioning**: Create appropriately tailored versions for each audience
4. **Consistency Architecture**: Ensure core facts remain consistent across versions
5. **Timing Orchestration**: Sequence communications for maximum effect
6. **Feedback Loop Design**: Build in mechanisms to detect and correct misunderstandings

### Expert Heuristic
"The same information, framed differently, creates completely different outcomes. The expert maintains a single source of truth while crafting multiple expressions optimized for each audience's context and concerns."
