# PCP-MC23: Resource Allocation Under Uncertainty

## Pattern Definition

### Trigger Condition
Activate when allocating limited resources (time, money, people, attention) across competing demands with uncertain outcomes or incomplete information about priorities.

### Core Procedure
1. **Resource Inventory**: Map all available and potentially available resources
2. **Demand Assessment**: Catalog competing demands and their uncertainty profiles
3. **Option Value Analysis**: Identify allocation choices that preserve future flexibility
4. **Fungibility Assessment**: Determine which resources can substitute for others
5. **Staged Commitment**: Design allocation that commits incrementally as uncertainty resolves
6. **Recovery Strategy**: Plan for reallocation when initial choices prove wrong

### Expert Heuristic
"Don't optimize for the expected case—optimize for learning. The expert allocates resources in ways that generate information about which demands actually matter, rather than betting everything on uncertain predictions."
